There are two versions of CoolProp Online:

1) `Full Version <index_full.html>`_ - Based on the javascript wrapper of CoolProp, and plotting powered by `Highcharts <http://highcharts.com>`_, enjoy a fully featured online version of CoolProp that allows for interactive thermodynamic property plots and generation of tabular data.
2) `Light Version <index_light.html>`_ - A simpler version of CoolProp Online, meant for users without javascript enabled, similar functionality, but a more limited interface.
